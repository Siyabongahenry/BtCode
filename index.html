If Statement

Language:
int n = 3;
int m = 3;
if( n == 0 )
  • Select A Condition
  • n == 3
  • n != 3
  • n > 3
  • n < 3
  • n <= 3
  • n >= 3
  • n > 0 && m < 4
  • n > 1 || m < 5
  • n > 0 && n < 1 || m < 5
  • n > 0 || n < 2 && m < 5
  • n < 0 || n < 2 && m < 5
{
Console.WriteLine(Hello world);
}
else
{
Console.WriteLine(Good bye world);
}
You can change some part of the code above but some you can't. Once the code starts executing, you cannot stop or pause the program. The program will stop once the if statement stop executing.

While Loop

Language:
int n = 3;
while( n < 3 )

  • n < 0
  • n < 5
  • n <= 5
  • n == 5
  • n > 1 || n < 5

//n++ increment n by 1

}

Do While Loop

Language:
//click the number to change the value
int n = 1;
do
{
Console.WriteLine(Hello World);
//n++ increment n by 1
n++;
}

  • Select A Condition
  • n < 0
  • n < 5
  • n <= 5
  • n == 5
  • n > 1 || n < 5
