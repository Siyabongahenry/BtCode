What to know about a for loop?

  • A for loop continues executing the code inside it if and only if the condition result to a true value.
  • If the condition result in a false value, the for loop is terminated, thus the code inside it is not execsuted.
  • The last condition in the for loop (e.g i++) is executed after the execution of the inside code.
