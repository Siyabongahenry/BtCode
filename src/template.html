How the logic worsk

Logic Condition
AND(&&) true && true = true true && false = false false && true = false false && false = false
OR(||) true || true = true true || false = true false || true = true false || false = false
NOT(!) !true = false !false = true

What to know about an if statement?

  • An if statement execute the code inside it, if and only if the condition result to a true value.
  • If the condition result in a false value, an if statement does not execute the code inside it
  • When working with boolean values, the if statement execute the code inside it, if and only if the boolean value is true.

The weird thing about an if statement

//The code below will run fine with no runtime error
string name;
if(name != null && name == "Siya" )
{
Console.WriteLine("hello world");
}
//The code below will cause a runtime error
string name;
if(name == "Siya" && name != null )
{
Console.WriteLine("hello world");
}

The secrete here is how an if statement solves the logic AND(&&). In an if statement, if the left hand side of an AND(&&) is false, the right hand side is not solved at all, a false value is returned immediately. The opposite applies an OR(||) logic, if the left hand side of an OR(||) is true the right hand side is not solved at all, a true value is immediately returned.

If you were to replace the the AND(&&) logic with an OR(||) in the above code, both code will results in a runtime error.

Understanding the above code will help you to prevent writing unnercessary nested if statements.s

FOR LOOP

/*Use this button to see how a for loop work*/
//i++ increment i by 1
for( int i= 0 ; i < 3 ; i++ ){
Console.WriteLine ( "Hello World" );
}

What to know about a for loop?

  • A for loop continues executing the code inside it if and only if the condition result to a true value.
  • If the condition result in a false value, the for loop is terminated, thus the code inside it is not executed.
  • The last condition in the for loop (e.g i++) is executed after the execution of the inside code.
